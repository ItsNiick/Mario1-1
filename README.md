# Mario1-1

Name: Steven Liu and Nicholas Shedd

Description: 
Super Mario Bro is a fun game where players get to be Mario as he runs through obstacles and collects coins. The goal is for Mario to reach to the castle to save Princess Peach from Bowser.

How to Play:
Begin play by pressing either Enter on the keyboard or numpad. Control Mario using left arrow or right arrow to move in the respective direction. Press up arrow or space bar to jump.
Make it from the start of the game to the end without falling off the map, and reach the castle to win.
